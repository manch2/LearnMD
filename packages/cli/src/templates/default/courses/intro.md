## What is LearnMD?

LearnMD is an open-source framework for creating interactive courses from Markdown files. It provides:

- 📝 **Markdown-based content** - Write courses in simple Markdown
- 🌍 **i18n Support** - Built-in internationalization
- 🎨 **Theming** - Customize the look and feel
- 🔌 **Plugins** - Extend functionality
