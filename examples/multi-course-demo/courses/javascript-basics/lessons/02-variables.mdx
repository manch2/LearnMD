# Variables en JavaScript

<Callout type="info">
  <Paragraph i18n="intro-vars">

## Declaring Variables

<Paragraph i18n="declaring">
  <en>In modern JavaScript, we use `let` and `const` to declare variables:</en>
  <es>En JavaScript moderno, usamos `let` y `const` para declarar variables:</es>
</Paragraph>

```javascript
// Using let - value can change
let age = 25;
age = 26; // ✓ Allowed

// Using const - value cannot change
const name = 'Alice';
// name = "Bob"; // ✗ Error!
```

## The Three Keywords

| Keyword | Reassignable | Scope    | Hoisted |
| ------- | ------------ | -------- | ------- |
| `var`   | Yes          | Function | Yes     |
| `let`   | Yes          | Block    | No      |
| `const` | No           | Block    | No      |

<Callout type="warning">
  <Paragraph i18n="best-practice">
    <en>
      **Best Practice:** Prefer `const` by default. Use `let` when you need to reassign. Avoid `var`
      in modern code.
    </en>
    <es>
      **Mejor práctica:** Prefiere `const` por defecto. Usa `let` cuando necesites reasignar. Evita
      `var` en código moderno.
    </es>
  </Paragraph>
</Callout>

## Variable Naming Rules

<Paragraph i18n="naming">
  <en>Variable names must follow these rules:</en>
  <es>Los nombres de variables deben seguir estas reglas:</es>
</Paragraph>

- ✅ Can contain letters, digits, underscores, and dollar signs
- ✅ Must begin with a letter, `$`, or `_`
- ✅ Are case-sensitive (`myVar` ≠ `myvar`)
- ✅ Cannot use reserved keywords

```javascript
// Valid variable names
let userName = 'John';
let $price = 99.99;
let _counter = 0;
let isActive = true;

// Invalid
// let 2name = "John";  // Cannot start with number
// let class = "JS";    // Cannot use reserved words
```
