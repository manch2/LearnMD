## What is JavaScript?

<Paragraph i18n="what-is-js">
  <en>
    JavaScript is a programming language that powers the web. Initially created to make web pages
    interactive, it has evolved into a versatile language used for:
  </en>
  <es>
    JavaScript es un lenguaje de programación que impulsa la web. Inicialmente creado para hacer
    páginas web interactivas, ha evolucionado hasta convertirse en un lenguaje versátil utilizado
    para:
  </es>
</Paragraph>

- 🌐 **Web Development** - Creating interactive websites
- 📱 **Mobile Apps** - Building cross-platform applications
- 🖥️ **Backend** - Server-side programming with Node.js
- 🎮 **Games** - Browser-based games and more

<Callout type="tip">

  </Paragraph>
</Callout>

## Your First Code

<Paragraph i18n="first-code">
  <en>
    Let's start with the classic "Hello, World!" program. In JavaScript, you can use `console.log()`
    to print messages:
  </en>
  <es>
    Comencemos con el clásico programa "¡Hola, Mundo!". En JavaScript, puedes usar `console.log()`
    para imprimir mensajes:
  </es>
</Paragraph>

```javascript
console.log('Hello, World!');
console.log('¡Hola, Mundo!');
```
