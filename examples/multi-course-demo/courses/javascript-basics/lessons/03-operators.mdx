# Operadores en JavaScript

<Callout type="info">
  <Paragraph i18n="intro-ops">
    <en>
      Operators allow you to perform operations on values. JavaScript has several types of operators
      for different purposes.
    </en>
    <es>
      Los operadores te permiten realizar operaciones con valores. JavaScript tiene varios tipos de
      operadores para diferentes propósitos.
    </es>
  </Paragraph>
</Callout>

## Arithmetic Operators

<Paragraph i18n="arith-intro">
  <en>Use arithmetic operators to perform mathematical calculations:</en>
  <es>Usa operadores aritméticos para realizar cálculos matemáticos:</es>
</Paragraph>

```javascript
let a = 10;
let b = 3;

console.log(a + b); // 13  (addition)
console.log(a - b); // 7   (subtraction)
console.log(a * b); // 30  (multiplication)
console.log(a / b); // 3.33... (division)
console.log(a % b); // 1   (remainder)
console.log(a ** b); // 1000 (exponentiation)
```

## Comparison Operators

<Paragraph i18n="comp-intro">
  <en>Comparison operators return `true` or `false`:</en>
  <es>Los operadores de comparación devuelven `true` o `false`:</es>
</Paragraph>

```javascript
let x = 5;
let y = '5';

console.log(x == y); // true  (loose equality)
console.log(x === y); // false (strict equality)
console.log(x != y); // false (loose inequality)
console.log(x !== y); // true  (strict inequality)
console.log(x > 3); // true
console.log(x <= 5); // true
```

<Callout type="warning">
  <Paragraph i18n="strict-vs-loose">
    <en>
      **Important:** Always use `===` and `!==` (strict) instead of `==` and `!=` (loose) to avoid
      unexpected type coercion.
    </en>
    <es>
      **Importante:** Siempre usa `===` y `!==` (estricto) en lugar de `==` y `!=` (laxo) para
      evitar conversiones de tipo inesperadas.
    </es>
  </Paragraph>
</Callout>

## Logical Operators

<Paragraph i18n="logical-intro">
  <en>Logical operators combine boolean expressions:</en>
  <es>Los operadores lógicos combinan expresiones booleanas:</es>
</Paragraph>

```javascript
let isAdult = true;
let hasLicense = false;

console.log(isAdult && hasLicense); // false (AND)
console.log(isAdult || hasLicense); // true  (OR)
console.log(!isAdult); // false (NOT)
```

## Operator Precedence

<Paragraph i18n="precedence">
  <en>JavaScript follows standard mathematical precedence:</en>
  <es>JavaScript sigue la precedencia matemática estándar:</es>
</Paragraph>

| Priority | Operators                          |
| -------- | ---------------------------------- |
| 1        | `()` (parentheses)                 |
| 2        | `**` (exponent)                    |
| 3        | `*`, `/`, `%`                      |
| 4        | `+`, `-`                           |
| 5        | `===`, `!==`, `<`, `>`, `<=`, `>=` |
| 6        | `&&`                               |
| 7        | `\|\|`                             |
