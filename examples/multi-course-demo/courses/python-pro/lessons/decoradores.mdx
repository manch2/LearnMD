# Decoradores

<Callout type="warning">
  Este tema requiere un buen entendimiento sobre cómo tratar funciones como objetos de primera clase.
</Callout>

<Paragraph i18n="intro">
  <en>
    Decorators allow us to wrap another function in order to extend the behavior of the wrapped function.
  </en>
  <es>
    Los decoradores nos permiten envolver una función para extender el comportamiento de la misma.
  </es>
</Paragraph>
