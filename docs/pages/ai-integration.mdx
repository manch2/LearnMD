## The LearnMD Course Creator Skill

<Paragraph i18n="skill-overview">
  <en>
    The `learnmd-course-creator` skill is a set of instructions and reference materials that teach AI agents how to use the LearnMD framework effectively. When an AI agent loads this skill, it gains the knowledge to:

    - Create new LearnMD projects from scratch
    - Scaffold courses with proper metadata and structure
    - Add lessons with i18n support, quizzes, callouts, and video embeds
    - Configure gamification (badges, points)
    - Set up PDF certificate generation
    - Sync lesson files with the course configuration
  </en>
  <es>
    El skill `learnmd-course-creator` es un conjunto de instrucciones y materiales de referencia que enseñan a los agentes de IA cómo usar el framework LearnMD efectivamente. Cuando un agente de IA carga este skill, obtiene el conocimiento para:

    - Crear nuevos proyectos LearnMD desde cero
    - Estructurar cursos con metadatos y estructura apropiados
    - Agregar lecciones con soporte i18n, quizzes, callouts y video embeds
    - Configurar gamificación (insignias, puntos)
    - Configurar generación de certificados PDF
    - Sincronizar archivos de lecciones con la configuración del curso
  </es>
</Paragraph>

## Non-Interactive Mode (Critical)

<Callout type="warning">
  <Paragraph i18n="non-interactive-critical">
    <en>
      **MANDATORY for AI Agents**: All CLI commands MUST use the `--non-interactive` (or `-n`) flag when executed by an AI agent. Without this flag, the CLI will display interactive prompts that cause the terminal to hang indefinitely.
    </en>
    <es>
      **OBLIGATORIO para Agentes de IA**: Todos los comandos del CLI DEBEN usar el flag `--non-interactive` (o `-n`) cuando se ejecutan por un agente de IA. Sin este flag, el CLI mostrará prompts interactivos que causan que la terminal se cuelgue indefinidamente.
    </es>
  </Paragraph>
</Callout>

<Paragraph i18n="non-interactive-examples">
  <en>
    **Correct (AI-safe):**
  </en>
  <es>
    **Correcto (seguro para IA):**
  </es>
</Paragraph>

```bash
# Create project
learnmd create my-course --non-interactive

# Add course
learnmd add course "Docker Fundamentals" --difficulty beginner --time "2 hours" --non-interactive

# Add lesson
learnmd add lesson "Containers 101" --course docker-fundamentals --non-interactive

# Sync
learnmd sync docker-fundamentals --non-interactive
```

<Paragraph i18n="non-interactive-wrong">
  <en>
    **Incorrect (will hang):**
  </en>
  <es>
    **Incorrecto (se colgará):**
  </es>
</Paragraph>

```bash
# ❌ These will prompt for user input and hang
learnmd create
learnmd add course "Docker"
learnmd sync docker-fundamentals
```

## AI Workflow: End-to-End Course Creation

<Paragraph i18n="workflow-intro">
  <en>
    Here is a complete example of how an AI agent creates a full course from a single user request:
  </en>
  <es>
    Aquí hay un ejemplo completo de cómo un agente de IA crea un curso completo a partir de una sola solicitud del usuario:
  </es>
</Paragraph>

### Step 1: Create the Course

```bash
learnmd add course "Kubernetes Essentials" \
  --difficulty intermediate \
  --time "4 hours" \
  --author "AI Assistant" \
  --description "Learn Kubernetes from pods to deployments" \
  --non-interactive
```

### Step 2: Add Lessons

```bash
learnmd add lesson "What is Kubernetes?" --course kubernetes-essentials --non-interactive
learnmd add lesson "Pods and Containers" --course kubernetes-essentials --non-interactive
learnmd add lesson "Services and Networking" --course kubernetes-essentials --non-interactive
learnmd add lesson "Deployments and Scaling" --course kubernetes-essentials --non-interactive
```

## Best Practices for AI-Generated Content

<Paragraph i18n="best-practices">
  <en>
    1. **Keep lessons focused**: One topic per lesson, 10-20 minutes reading time maximum
    2. **Use quizzes strategically**: Place them after key concepts, not after every paragraph
    3. **Add visual variety**: Mix videos, callouts, code blocks, and text
    4. **Always sync**: After creating or modifying files, run `learnmd sync <slug> --non-interactive`
    5. **Non-interactive is mandatory**: Always append `--non-interactive` to all CLI commands
    6. **Bilingual content**: Use `<Title>` and `<Paragraph>` i18n wrappers for all text content
    7. **Descriptive frontmatter**: Always include `title`, `description`, and `duration` in lesson frontmatter
  </en>
  <es>
    1. **Mantén las lecciones enfocadas**: Un tema por lección, máximo 10-20 minutos de lectura
    2. **Usa quizzes estratégicamente**: Colócalos después de conceptos clave, no después de cada párrafo
    3. **Agrega variedad visual**: Mezcla videos, callouts, bloques de código y texto
    4. **Siempre sincroniza**: Después de crear o modificar archivos, ejecuta `learnmd sync <slug> --non-interactive`
    5. **No interactivo es obligatorio**: Siempre agrega `--non-interactive` a todos los comandos del CLI
    6. **Contenido bilingüe**: Usa wrappers i18n `<Title>` y `<Paragraph>` para todo el contenido de texto
    7. **Frontmatter descriptivo**: Siempre incluye `title`, `description` y `duration` en el frontmatter de las lecciones
  </es>
</Paragraph>
