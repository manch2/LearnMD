## Basic Example

```typescript
import { defineConfig } from '@learnmd/core';
import { BadgesPlugin } from '@learnmd/plugin-badges';
import { PDFPlugin } from '@learnmd/plugin-pdf';

export default defineConfig({
  title: 'My Learning Platform',
  description: 'Interactive courses powered by LearnMD',
  defaultLanguage: 'en',
  availableLanguages: ['en', 'es'],

  theme: {
    primaryColor: '#3b82f6',
    secondaryColor: '#14b8a6',
    fontFamily: '"Inter", system-ui, sans-serif',
    headingFontFamily: '"Space Grotesk", "Inter", system-ui, sans-serif',
    contentMaxWidth: '72rem',
    logoText: 'My Docs',
    darkMode: {
      enabled: true,
      backgroundColor: '#0f172a',
      surfaceColor: '#1e293b',
      textColor: '#e2e8f0',
      mutedTextColor: '#94a3b8',
    },
  },

  gamification: {
    points: {
      lessonCompletion: 100,
      quizPassed: 10,
      quizPerfectScore: 25,
    },
  },

  navigation: [
    { label: { en: 'Getting Started', es: 'Primeros Pasos' }, path: '/getting-started' },
    { label: { en: 'Plugins', es: 'Plugins' }, path: '/plugins' },
  ],

  customPages: [
    { path: '/getting-started', componentPath: 'pages/getting-started.mdx' },
    { path: '/plugins', componentPath: 'pages/plugins.mdx' },
  ],

  plugins: [
    new BadgesPlugin([
      {
        id: 'global-500',
        name: 'XP 500',
        description: 'Reach a global score of 500.',
        icon: '🏆',
        criteria: { type: 'global_score', score: 500 },
      },
    ]),
    new PDFPlugin(),
  ],
});
```

## Root Options

| Option | Type | Notes |
|--------|------|-------|
| `title` | `string` | Brand title used by the app shell |
| `description` | `string` | Project description |
| `defaultLanguage` | `string` | Initial language |
| `availableLanguages` | `string[]` | Supported languages |
| `basePath` | `string` | Deployment base path |
| `storagePrefix` | `string` | Prefix for LearnMD storage |
| `enableAnalytics` | `boolean` | Reserved analytics flag |
| `theme` | `ThemeConfig` | Visual customization |
| `gamification` | `false \| GamificationConfig` | Global score and achievement rules |
| `navigation` | `NavigationItem[]` | Header links |
| `customPages` | `CustomPage[]` | Static MDX routes |
| `plugins` | `Plugin[]` | Registered plugins |

<Callout type="info">
  `enableGamification` is no longer the practical switch. In beta.3 the runtime uses `gamification: false` to disable the built-in gamification plugin.
</Callout>

## Theme Configuration

The theme now exposes the options that are actually consumed by the default theme:

| Option | Type |
|--------|------|
| `primaryColor` | `string` |
| `secondaryColor` | `string` |
| `backgroundColor` | `string` |
| `surfaceColor` | `string` |
| `textColor` | `string` |
| `mutedTextColor` | `string` |
| `borderColor` | `string` |
| `fontFamily` | `string` |
| `headingFontFamily` | `string` |
| `borderRadius` | `number \| string` |
| `contentMaxWidth` | `number \| string` |
| `logoText` | `string` |
| `darkMode` | `boolean \| object` |

```typescript
theme: {
  primaryColor: '#2563eb',
  secondaryColor: '#0f766e',
  fontFamily: '"Inter", system-ui, sans-serif',
  headingFontFamily: '"Space Grotesk", "Inter", system-ui, sans-serif',
  contentMaxWidth: '72rem',

    enabled: true,
    backgroundColor: '#0f172a',
    surfaceColor: '#1e293b',
    textColor: '#e2e8f0',
    mutedTextColor: '#94a3b8',
  },
}
```

## Gamification Configuration

Gamification now controls the global public score and achievements. Badges remain a separate plugin.

```typescript
gamification: {
  points: {
    lessonCompletion: 100,
    quizPassed: 10,
    quizPerfectScore: 25,
  },
  streak: {
    enabled: true,
    bonusByDays: {
      7: 50,
      30: 200,
    },
  },
}
```

To disable the built-in gamification plugin completely:

```typescript
gamification: false
```

## Plugins

Plugins are the extension mechanism for behavior and UI. If a plugin is not registered in `plugins`, its UI must not appear in the application.

```typescript
plugins: [
  new BadgesPlugin([
    {
      id: 'score-1000',
      name: 'XP 1000',
      description: 'Reach a global score of 1000.',
      icon: '🚀',
      criteria: { type: 'global_score', score: 1000 },
    },
  ]),
  new PDFPlugin(),
]
```

## Custom Pages

`customPages` maps app routes to MDX files. In beta.3 those pages share the same content container as the home content, so spacing and readable width stay consistent.

```typescript
customPages: [
  { path: '/getting-started', componentPath: 'pages/getting-started.mdx' },
  { path: '/plugins', componentPath: 'pages/plugins.mdx' },
]
```

## Migration Notes for Beta.3

1. Replace old `pointsPerLesson` and `pointsPerQuiz` with `gamification.points.lessonCompletion` and `gamification.points.quizPassed`.
2. Use `gamification: false` instead of `enableGamification: false`.
3. Keep badges in `@learnmd/plugin-badges`; the core gamification plugin owns score and achievements, not badge criteria.
4. Register `PDFPlugin` explicitly if you want certificate actions in the UI.
