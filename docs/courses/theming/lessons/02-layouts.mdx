  </es>

## Default Pattern / Patrón por defecto

<Paragraph i18n="pattern">
  <en>
    Lessons use the course viewer layout, while static pages such as home and custom pages render through `ContentPage`. That gives them the same readable width and spacing rules.
  </en>
  <es>
    Las lecciones usan el layout del course viewer, mientras que las páginas estáticas como el home y las páginas personalizadas se renderizan mediante `ContentPage`. Eso les da las mismas reglas de ancho legible y espaciado.
  </es>
</Paragraph>

```tsx
<MainLayout>
  <Header />
  <ContentPage>
    <MyCustomPage />
  </ContentPage>
</MainLayout>
```

<Callout type="success">
  <Paragraph i18n="success">
    <en>
      Keep layout concerns in the theme or scaffold, not inside each content page. That is the easiest way to avoid width and spacing drift over time.
    </en>
    <es>
      Mantén las decisiones de layout en el theme o en el scaffold, no dentro de cada página de contenido. Esa es la forma más simple de evitar que el ancho y el espaciado se desalineen con el tiempo.
    </es>
  </Paragraph>
</Callout>
