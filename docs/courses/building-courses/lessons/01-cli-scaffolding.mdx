## Common Commands / Comandos comunes

<Paragraph i18n="commands">
  <en>
    Use `learnmd create` to scaffold a project, `learnmd add course` and `learnmd add lesson` to expand content, and `learnmd sync` when you need to reconcile MDX files with `learnmd.json`.
  </en>
  <es>
    Usa `learnmd create` para generar un proyecto, `learnmd add course` y `learnmd add lesson` para ampliar contenido, y `learnmd sync` cuando necesites reconciliar archivos MDX con `learnmd.json`.
  </es>
</Paragraph>

```bash
npx learnmd add lesson "My New Lesson" --course building-courses
```

<Callout type="info">
  <Paragraph i18n="sync-note">
    <en>
      You can also create MDX files manually and then run `learnmd sync` to rebuild the course metadata.
    </en>
    <es>
      También puedes crear archivos MDX manualmente y luego ejecutar `learnmd sync` para reconstruir los metadatos del curso.
    </es>
  </Paragraph>
</Callout>
