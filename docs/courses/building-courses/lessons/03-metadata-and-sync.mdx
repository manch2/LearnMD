<Paragraph i18n="intro">
  <en>
    Each course keeps a `learnmd.json` file with high-level metadata and lesson ordering. This gives the runtime a quick index without parsing every MDX file first.
  </en>
  <es>
    Cada curso mantiene un archivo `learnmd.json` con metadatos de alto nivel y el orden de las lecciones. Esto le da al runtime un índice rápido sin tener que parsear primero todos los archivos MDX.
  </es>
</Paragraph>

## What Lives There / Qué vive ahí

<Paragraph i18n="metadata-fields">
  <en>
    Typical fields include the course `title`, `description`, difficulty, estimated time, and the ordered `lessons` array. The file acts as the course manifest consumed by the app and the CLI.
  </en>
  <es>
    Los campos típicos incluyen el `title`, la `description`, la dificultad, el tiempo estimado y el arreglo ordenado de `lessons`. El archivo actúa como el manifiesto del curso consumido por la app y el CLI.
  </es>
</Paragraph>

```bash
npx learnmd sync building-courses
```

<Callout type="warning">
  <Paragraph i18n="warning">
    <en>
      Run `sync` after adding, renaming, or reordering lesson files manually so the JSON manifest stays accurate.
    </en>
    <es>
      Ejecuta `sync` después de agregar, renombrar o reordenar manualmente archivos de lección para que el manifiesto JSON siga siendo preciso.
    </es>
  </Paragraph>
</Callout>
