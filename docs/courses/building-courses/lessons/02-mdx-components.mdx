## Built-in Components / Componentes integrados

<Paragraph i18n="components">
  <en>
    Common components include `Callout`, `Quiz`, `Progress`, `VideoEmbed`, `Title`, and `Paragraph`. Together they let you create more guided and interactive lessons than plain markdown alone.
  </en>
  <es>
    Los componentes comunes incluyen `Callout`, `Quiz`, `Progress`, `VideoEmbed`, `Title` y `Paragraph`. Juntos te permiten crear lecciones más guiadas e interactivas que con markdown plano.
  </es>
</Paragraph>

```mdx
<Callout type="info">
  This is an informative callout!
</Callout>
```

```mdx
<Quiz
  id="sample-quiz"
  questions={[
    {
      id: 'q1',
      type: 'multiple-choice',
      question: { en: 'What is 1 + 1?', es: '¿Cuánto es 1 + 1?' },
      options: [
        { id: 'a', label: { en: '1', es: '1' } },
        { id: 'b', label: { en: '2', es: '2' } },
      ],
      correctAnswer: 'b',
    },
  ]}
/>
```
