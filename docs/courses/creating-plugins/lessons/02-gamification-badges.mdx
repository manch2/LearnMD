<Paragraph i18n="intro">
  <en>
    LearnMD separates score logic from badge logic. The built-in gamification plugin owns `globalScore`, achievements, and streaks, while `@learnmd/plugin-badges` evaluates badge criteria.
  </en>
  <es>
    LearnMD separa la lógica de puntaje de la lógica de insignias. El plugin integrado de gamificación es dueño del `globalScore`, los logros y las rachas, mientras `@learnmd/plugin-badges` evalúa criterios de badges.
  </es>
</Paragraph>

## Points and Score / Puntos y score

<Paragraph i18n="points">
  <en>
    Points are configured in `gamification.points.*` and rolled into the public profile score. This makes the score reusable by badges, leaderboards, or any future plugin that needs a stable reputation signal.
  </en>
  <es>
    Los puntos se configuran en `gamification.points.*` y se acumulan en el puntaje público del perfil. Esto hace que el score sea reutilizable por badges, leaderboards o cualquier plugin futuro que necesite una señal estable de reputación.
  </es>
</Paragraph>

```typescript
export default defineConfig({
  gamification: {
    points: {
      lessonCompletion: 100,
      quizPassed: 10,
      quizPerfectScore: 25,
    },
  },
  plugins: [
    new BadgesPlugin([
      {
        id: 'score-500',
        name: { en: 'Momentum', es: 'Impulso' },
        description: { en: 'Reach 500 global points.', es: 'Alcanza 500 puntos globales.' },
        icon: '??',
        criteria: { type: 'global_score', score: 500 },
      },
    ]),
  ],
});
```
