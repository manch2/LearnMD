## Core API / API principal

<Paragraph i18n="api">
  <en>
    A plugin implements `onLoad`, receives a `PluginContext`, and can register hooks or UI slot components. Optional `onUnload` logic can clean up resources if your plugin needs it.
  </en>
  <es>
    Un plugin implementa `onLoad`, recibe un `PluginContext` y puede registrar hooks o componentes para slots de UI. La lógica opcional `onUnload` puede limpiar recursos si tu plugin lo necesita.
  </es>
</Paragraph>

```typescript
import { BasePlugin, type PluginContext } from '@learnmd/core';

export class LoggerPlugin extends BasePlugin {
  constructor() {
    super('logger', '1.0.0');
  }

  onLoad(ctx: PluginContext) {
    ctx.registerHook('lesson:complete', async (payload) => {
      console.log('Lesson completed', payload);
    });
  }
}
```

<Callout type="info">
