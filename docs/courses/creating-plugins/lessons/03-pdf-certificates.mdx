# PDF Certificates

The PDF plugin owns certificate generation in LearnMD beta.3.

## Features

-   Certificate downloads are available only when `new PDFPlugin()` is registered.
-   The plugin injects its UI into the `profile:courseActions` slot.
-   You can customize the default signature and provide a custom React template.

## Usage

Register the plugin in `learnmd.config.ts`:

```typescript
import { defineConfig } from '@learnmd/core';
import { PDFPlugin } from '@learnmd/plugin-pdf';

export default defineConfig({
  plugins: [
    new PDFPlugin({
      signature: 'LearnMD Team',
    }),
  ],
});
```

## How it works

The plugin registers a UI action for completed courses in the profile. If the plugin is missing,
no certificate button is rendered and the user has no alternative certificate flow in the default
UI.

## Custom template

```typescript
import React from 'react';
import { PDFPlugin } from '@learnmd/plugin-pdf';

const CertificateTemplate = (
  <div style={{ width: '11in', height: '8.5in', padding: '48px' }}>
    <h1>Certificate of Completion</h1>
    <p>Issued by LearnMD.</p>
  </div>
);

new PDFPlugin({
  signature: 'Lead Instructor',
  template: CertificateTemplate,
});
```

<Callout type="warning">
  The default theme only shows certificate actions when the plugin is present and the course is
  completed.
</Callout>
