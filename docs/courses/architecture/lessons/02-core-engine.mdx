  </en>
  <es>
    `@learnmd/core` es el corazón del runtime del framework. Expone el provider, helpers de routing, storage, utilidades de parsing, orquestación de gamificación y el registro de plugins.
  </es>
</Paragraph>

## Main Responsibilities / Responsabilidades principales

<Paragraph i18n="responsibilities">
  <en>
    The core package parses MDX metadata, initializes the shared app state, persists progress, and coordinates hooks between official and custom plugins.
  </en>
  <es>
    El paquete core parsea metadatos MDX, inicializa el estado compartido de la app, persiste el progreso y coordina hooks entre plugins oficiales y personalizados.
  </es>
</Paragraph>

```typescript
import { createLearnMD } from '@learnmd/core';

const app = createLearnMD({
  defaultLanguage: 'en',
  gamification: {
    points: {
      lessonCompletion: 100,
    },
  },
});
```

<Callout type="warning">
  <Paragraph i18n="warning">
    <en>
      Prefer extending the runtime through plugins before modifying the core directly. Core changes should benefit the framework as a whole.
    </en>
    <es>
      Prefiere extender el runtime mediante plugins antes de modificar el core directamente. Los cambios en el core deben beneficiar al framework completo.
    </es>
  </Paragraph>
</Callout>
