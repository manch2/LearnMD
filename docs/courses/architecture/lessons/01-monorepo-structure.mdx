# Monorepo Structure

LearnMD uses a monorepo structure to manage its various packages and examples efficiently. This is powered by [Turborepo](https://turbo.build/).

## Benefits of Monorepo

- **Code Sharing**: Share types, components, and utilities across packages easily.
- **Consistent Tooling**: Unified linting, testing, and building configurations.
- **Simplified Dependency Management**: All packages use the same version of shared dependencies.

## Key Directories

### `packages/`

This is where the core logic of LearnMD lives:

-   `core/`: The foundational engine.
-   `cli/`: The developer command-line interface.
-   `default-theme/`: The standard look and feel.
-   `plugins/`: Official extensions.

### `docs/`

The site you are currently viewing! It's a LearnMD project itself, demonstrating how to use the framework for documentation.

### `examples/`

Ready-to-use templates and demonstration projects to help you get started quickly.
